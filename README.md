# 🥚 Quail aviculture 🐣
Quail aviculture automation on Arduino uno. developed on [platformio](https://platformio.org/).

### 📌 Pin Defines
#### INPUT Pins:  
`#2` Setter tempeture data  
`#4` Hatcher tempeture data

#### OUTPUT Pins:
`#7` Heater (Lamp)  
`#12` Fan  
`#8` Termometer check

### 🌡️ Tempeture
Setter:  
min `37.5 c` - max `37.7 c`

Hatcher:  
min `37.15 c` - max `37.25 c`
